# ConnEthics Website

A modern Next.js website for ConnEthics, built with TypeScript, Tailwind CSS, and React. ConnEthics specializes in building ethical connections in competitive environments through self-sovereign identity, competitive intelligence, and strategic product leadership.

## 🚀 Features

- **Self-Sovereign Identity** - SSI integration and deployment services
- **Ecosystem Cartography** - Competitive intelligence and economic warfare analysis
- **Trust & Identity Deployment** - Verifiable credentials and certification
- **Product Leadership** - Strategic roadmap alignment and OKR implementation
- **Product Management** - Team scaling and release process optimization

## 🛠️ Tech Stack

- **Framework:** Next.js 15 with App Router
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Icons:** Heroicons
- **Deployment:** Vercel-ready
- **Development:** Hot reloading with Turbopack

## 📁 Project Structure

```
src/
├── app/                 # Next.js App Router pages
│   ├── about/          # About page
│   ├── contact/        # Contact form page
│   ├── insights/       # Blog/insights page
│   ├── services/       # Services overview page
│   ├── layout.tsx      # Root layout
│   └── page.tsx        # Homepage
└── components/         # Reusable React components
    ├── Navigation.tsx  # Header navigation
    └── Footer.tsx      # Site footer
```

## 🚀 Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Open your browser:**
   Visit [http://localhost:3000](http://localhost:3000) to see the website.

## 📝 Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run ESLint

## 🎨 Design System

The website follows ConnEthics' brand guidelines:
- **Primary Color:** Blue-900 (#1e3a8a)
- **Typography:** Inter font family
- **Design:** Clean, professional, trust-building
- **Approach:** Mobile-first responsive design

## 📄 Pages

- **Homepage** - Hero section, services overview, and call-to-actions
- **About** - Company mission, values, and story
- **Services** - Detailed service offerings and methodologies
- **Insights** - Thought leadership content and articles
- **Contact** - Contact form and company information

## 🔧 Customization

The website is built with modularity in mind. Key areas for customization:

- **Brand Colors:** Update in `globals.css` and component files
- **Content:** Modify page content in respective `page.tsx` files
- **Navigation:** Update links in `Navigation.tsx`
- **Footer:** Modify footer content in `Footer.tsx`

## 🚀 Deployment

This project is optimized for deployment on Vercel:

1. **Push to GitHub**
2. **Connect to Vercel**
3. **Deploy automatically**

For other platforms, run `npm run build` to generate the production build.
